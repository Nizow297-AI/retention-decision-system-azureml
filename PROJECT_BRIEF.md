# Project Brief: Retention Decision System

## Business Problem
Customer churn is a major challenge in banking and retail-focused financial services. Losing customers reduces revenue, weakens long-term value, and increases acquisition costs. Businesses need to identify customers at risk of churning and determine which retention strategy is most effective.

## Project Goal
Build an end-to-end retention decision system that:
1. Predicts which customers are likely to churn
2. Identifies the high-risk segment
3. Evaluates which retention strategy is most effective using A/B testing logic
4. Produces business recommendations for targeted retention action

## Project Objectives
- Build a churn prediction model using structured customer data
- Evaluate model performance using classification metrics
- Define a threshold for identifying high-risk customers
- Analyze A/B testing data to compare intervention effectiveness
- Recommend the best retention strategy for high-risk customers
- Present findings in a professional GitHub-ready format

## Key Business Questions
1. Which customers are most likely to churn?
2. What factors are associated with churn?
3. Which intervention performs better?
4. How should a business target retention efforts?
